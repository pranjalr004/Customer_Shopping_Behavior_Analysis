# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer shopping behavior using Python, PostgreSQL, SQL, and Power BI to uncover customer preferences, purchasing trends, and business insights.

The project follows a complete Data Analytics workflow:

* Data Collection & Loading
* Exploratory Data Analysis (EDA)
* Data Cleaning & Preparation
* SQL-Based Business Analysis
* Interactive Power BI Dashboard Development
* Business Reporting & Presentation (Gamma AI)

The objective is to transform raw customer shopping data into actionable insights that support data-driven decision-making.

The dataset contains customer shopping information including:

# 🗄️ SQL Analysis

## Revenue by Gender

### Query

```sql
SELECT gender,
       SUM(purchase_amount) AS revenue
FROM customer
GROUP BY gender;
```

### Result

| Gender | Revenue |
| ------ | ------: |
| Female |  75,191 |
| Male   | 157,890 |

### Insight

Male customers generated more than twice the revenue of female customers, making them the dominant revenue-contributing segment.

---

## Top 5 Highest Rated Products

### Query

```sql
SELECT item_purchased,
       ROUND(AVG(review_rating::numeric),2) AS average_rating
FROM customer
GROUP BY item_purchased
ORDER BY AVG(review_rating) DESC
LIMIT 5;
```

### Result

| Product | Rating |
| ------- | -----: |
| Gloves  |   3.86 |
| Sandals |   3.84 |
| Boots   |   3.82 |
| Hat     |   3.80 |
| Skirt   |   3.78 |

### Insight

Gloves received the highest average customer rating, followed closely by Sandals and Boots, indicating strong customer satisfaction within these product categories.
